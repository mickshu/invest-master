### 市场特有分析考量

#### 🇺🇸 美股特有考量
- **会计准则**：美股采用 US GAAP，部分中概股采用 IFRS。财务指标口径可能与A股不同。
- **报告周期**：美股财年可能非日历年（如苹果财年截至9月底），需注意 `financial_date` 与 `end_date` 的区别。
- **SEC文件**：10-K（年报）、10-Q（季报）、8-K（重大事件）是核心信息源，MCP `get_sec_filings` 可获取。
- **机构持仓**：13F filings 每季度披露，是判断"华尔街关注度"的关键数据。
- **回购文化**：美股回购远多于A股，计算 Owner Earnings 时需扣除回购支出。
- **汇率风险**：中国投资者买入美股需承担人民币/美元汇率波动。
- **ADR vs 普通股**：中概股多为 ADR，需关注退市风险和存托银行费用。

#### 🇭🇰 港股特有考量
- **会计准则**：港股采用 HKFRS，与A股 CAS 和美股 US GAAP 均有差异。
- **货币单位**：港股以港币计价，分析时需统一换算为人民币或美元做跨市场对比。
- **AH股溢价**：同一公司在A股和港股可能存在显著价差，AH溢价指数是重要参考。
- **港股通资金**：南向资金流向是港股重要风向标。
- **做空机制**：港股做空更为普遍，需关注做空报告和借股利率。
- **流动性风险**：部分港股成交量极低，买卖价差大，流动性折价需纳入考量。
- **红利税**：通过港股通买入 H 股红利税 20%，红筹股 28%，影响实际股息收益。

## 多大师深度分析框架

### 第一阶段：商业本质（巴菲特 + 段永平 + 费雪主导）

> "买股票就是买公司。如果说不清它靠什么赚钱，就不要买。" —— 段永平

1. **能力圈/本分检验**：能否一段话说清赚钱方式？（巴菲特 + 段永平）
2. **护城河判断**：五种护城河类型 + 趋势（变宽/稳定/变窄）（巴菲特）
3. **定价权测试**：能否涨价不流失客户？（巴菲特 + 芒格 + 段永平）
4. **商业模式简洁性**：是不是"简单、可理解、可持续"的生意？（段永平）
5. **增长潜力**：市场潜力是否足够支撑多年增长？（费雪）
6. **"对的事情"检验**：公司做的事本质上是否"对"？（段永平）

### 第二阶段：分类与故事（林奇 + 费雪主导）

> "买之前，先搞清楚这是什么类型的股票。" —— 林奇

1. **六类分类**：Slow Grower / Stalwart / Fast Grower / Cyclical / Turnaround / Asset Play（林奇）
2. **两分钟演练**：能否两分钟内讲清楚为什么这只股票会涨？（林奇）
3. **15点评分**：商业特征(1-4) / 盈利能力(5-8) / 人的因素(9-12) / 竞争地位(13-15)（费雪）
4. **闲聊法信息**：客户/供应商/竞争对手/前员工怎么说？（费雪）

### 第三阶段：财务与估值（巴菲特 + 林奇 + 段永平主导）

1. **Owner Earnings** = 净利润 + 折摊 - 维持性资本支出 - 营运资本增加（巴菲特）
2. **PEG决策**：PEG < 0.5 极度吸引 → PEG > 2.0 过贵（林奇）
3. **ROIC** 10年平均目标 >15%；现金转化率目标 >90%（巴菲特 + 芒格）
4. **"睡得着觉"估值**：3年后回头看便不便宜？（段永平）
5. **安全边际分级**：高确定性20-30% / 一般优秀30-40% / 有不确定性40-50%（巴菲特）

### 第四阶段：风险与认知（芒格 + 马克斯主导）

> "先问如何亏钱，再问如何赚钱。" —— 芒格

1. **逆向思维**：3条失败路径是什么？是否可避免？（芒格）
2. **25种认知偏差检查**：确认偏差/承诺偏差/权威偏差/可得性偏差/剥夺偏差（芒格）
3. **第二层次思维**：共识是什么？为什么可能是错的？（马克斯）
4. **风险 = 永久资本损失**：概率和幅度？（马克斯）
5. **周期定位**：经济/利润/信贷/心理/估值周期当前位置？（马克斯）
6. **Stop Doing检查**：公司/管理层/投资者自身是否在做不该做的事？（段永平）

### 第五阶段：宏观与配置（达利欧 + 马克斯主导）

> "不理解债务周期，就无法理解经济。" —— 达利欧

1. **经济机器**：驱动资产价格的核心因果链（达利欧）
2. **周期定位**：短期债务周期(5-8年) + 长期债务周期(75-100年)（达利欧）
3. **范式识别**：当前范式是什么？可能如何转变？（达利欧）
4. **全天候检验**：增长/通胀四象限是否都能存活？（达利欧）
5. **周期定位表**：深度低谷→早期复苏→中期→晚期→顶峰狂热（马克斯）

### 规则3：关键数值交叉校验（三市场通用）

提取以下关键数据后，必须做合理性校验：

| 指标 | 校验规则 | 不合理时的处理 |
|------|---------|--------------|
| 股价 | 与近5日涨跌幅推算值偏差<10% | 重新查询 |
| PE(TTM) | 与 PB/ROE 关系 ≈ PE = PB/ROE | 偏差>25%时重新查询 |
| PB | 与 市值/净资产 推算值偏差<20% | 重新查询 |
| 市值 | 与 股价×总股本 推算值偏差<20% | 重新查询 |
| ROE | 与 净利润/净资产 年化值偏差<30% | 重新查询 |

**A股交叉校验公式：** `PE ≈ PB / ROE`（例如 PB=10.78，ROE=20.33%，则 PE ≈ 10.78/0.2033 ≈ 53.0，实际 53.42 ✅）

**美股交叉校验：**
1. MCP `get_income_statements` 返回的 net_income 与 `us_fina_indicator` 的 parent_holder_netprofit 应大致一致
2. `us_daily` 返回的 PE/PB 与 `us_fina_indicator` 的 ROE 应满足 `PE ≈ PB/ROE`
3. 市值校验：`us_daily.total_mv` 应与 `股价 × 总股本` 吻合
4. 注意美股市值单位通常为美元，与A股亿元人民币差8个量级

**港股交叉校验：**
1. `hk_daily` 的 close 价格与 neodata 查询的实时价偏差应<5%（排除盘后变动）
2. 如有AH股上市，AH溢价应在合理范围（通常 0.5-2.0）

### 规则7：美股财务数据特别注意事项

1. **金额单位**：MCP financial-datasets 返回的金额单位默认为美元原始值；finance-data-retrieval `us_fina_indicator` 返回的金额为美元原始值。在报告中统一换算时需确认单位。
2. **财年差异**：美股公司财年可能非日历年。例如苹果(AAPL)财年截至9月底，`us_fina_indicator` 中 `end_date` 为 `20240928` 对应 Q4 年报。在引用财务数据时，必须注明"FY2024(截至2024/09)"而非简单的"2024年"。
3. **GAAP vs Non-GAAP**：美股公司普遍报告 Non-GAAP 指标（adjusted earnings）。分析中应优先使用 GAAP 数据，如有重大差异需在报告中说明。
4. **回购影响**：美股公司大量回购导致流通股减少，EPS增长可能不完全反映实际盈利增长。需同时关注净利润绝对值和每股收益。
5. **负权益**：由于大量回购，部分美股公司（如麦当劳、波音）净资产为负，导致PE/PB失真。此时应改用 EV/EBITDA 或其他指标。

### 规则8：港股财务数据特别注意事项

1. **报告币种**：港股财报可能以港币、人民币或美元报告（由 `currency` 字段标识），分析时必须统一换算。
2. **半年报**：港股公司通常发布半年报而非季报，财务数据更新频率低于A股和美股。
3. **同股不同权**：部分港股公司（如美团、小米）采用同股不同权架构，投票权与经济利益不对等，影响管理层评估。
4. **港股通红利税**：通过港股通买入的 H 股红利税 20%，红筹股最高 28%，实际股息收益低于名义股息率。
