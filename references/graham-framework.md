# 格雷厄姆投资思维框架

> "市场短期是投票机，长期是称重机。"

## 快速筛选（8问）

| # | 维度 | 问题 | 否 = 红旗 |
|---|------|------|-----------|
| 1 | **安全边际** | 当前价格是否显著低于内在价值？ | 否 = 等待安全边际出现 |
| 2 | **内在价值** | 能否用定量方法估算内在价值？ | 不能 = 看不懂，跳过 |
| 3 | **财务稳健** | 流动比率>2且长期债务<净流动资产？ | 否 = 财务风险偏高 |
| 4 | **盈利稳定** | 过去10年是否每年都有盈利？ | 否 = 盈利不够稳定 |
| 5 | **股息记录** | 是否连续20年以上支付股息？ | 否 = 缺乏股东回报意识 |
| 6 | **盈利增长** | 过去10年EPS是否增长至少1/3？ | 否 = 增长不足 |
| 7 | **适度PE** | PE(TTM)是否<15？ | 过高 = 价格不够便宜 |
| 8 | **适度PB** | PB是否<1.5？ | 过高 = 资产溢价过高 |

> 格雷厄姆的定量筛选是"守门员"——不满足基本标准的公司，无论故事多好，都不值得深入分析。

## 深度分析框架

### 第1步：安全边际信仰检验

- **价格≠价值**：是否真正理解市场先生只是报价服务，不是指导服务？
- **安全边际>33%**：格雷厄姆建议的最低安全边际是33%（内在价值的2/3以下买入）
- **情绪控制**：能否在市场恐慌时买入，在市场贪婪时卖出？
- **投机vs投资**：当前行为是基于事实分析的投资，还是基于市场情绪波动的投机？

> "投资操作是基于彻底的分析，承诺本金安全和满意回报。不满足这些条件的操作就是投机。"

### 第2步：内在价值计算

**格雷厄姆公式：**
```
V = EPS × (8.5 + 2g) × (4.4 / Y)
其中：EPS = 过去12个月每股收益
     g = 预期年增长率(%)
     8.5 = 零增长公司的合理PE
     4.4 = 1962年美国AAA债券利率
     Y = 当前AAA债券利率
```

**简化公式（保守版）：**
```
V = sqrt(22.5 × EPS × BVPS)
其中：BVPS = 每股净资产
```

**净流动资产价值（NCAV）：**
```
NCAV = 流动资产 - 总负债
NCAV/股 = NCAV / 流通股数
```
格雷厄姆最青睐的股票：股价 < NCAV/股（即"用低于清算价值的价格买入"）

### 第3步：定量筛选清单

| 标准 | 格雷厄姆要求 | 计算方法 |
|------|-------------|---------|
| 流动比率 | > 2.0 | 流动资产/流动负债 |
| 长期债务 | < 净流动资产 | 长期债务 < (流动资产-流动负债) |
| EPS稳定性 | 10年无亏损 | 检查过去10年EPS |
| 股息连续性 | 20年不间断 | 检查股息历史 |
| EPS增长 | 10年增长≥33% | (最新EPS-10年前EPS)/10年前EPS |
| PE | < 15 | 股价/EPS(TTM) |
| PB | < 1.5 | 股价/每股净资产 |
| PE×PB | < 22.5 | PE × PB < 22.5 |

### 第4步：市场先生理解

- 市场先生每天给你报价，你可以接受或拒绝
- 市场先生的情绪波动创造了买入和卖出的机会
- 不要让市场先生的情绪影响你的判断
- 利用市场先生的疯狂，而非被其利用

### 第5步：风险与卖出

**格雷厄姆的风险定义：** 永久性的资本损失
- 不是波动性，而是内在价值的永久性下降
- 最大的风险是支付了过高的价格

**卖出标准：**
1. 价格涨至内在价值以上（安全边际消失）
2. 发现计算内在价值时的关键假设错误
3. 公司基本面发生永久性恶化
