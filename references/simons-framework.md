# 西蒙斯投资思维框架

> "市场不是不可预测的，只是需要正确的模型。"

## 快速筛选（7问）

| # | 维度 | 问题 | 否 = 红旗 |
|---|------|------|-----------|
| 1 | **数据质量** | 使用的数据是否干净、完整、无偏差？ | 否 = 垃圾进垃圾出 |
| 2 | **统计显著性** | 发现的规律是否在统计上显著(p<0.05)？ | 否 = 可能是随机噪声 |
| 3 | **样本外测试** | 模型在未见过的数据上是否仍然有效？ | 否 = 过拟合 |
| 4 | **交易成本** | 策略收益是否足以覆盖交易成本？ | 否 = 理论可行但实际亏损 |
| 5 | **容量限制** | 策略容量是否足以容纳计划资金？ | 否 = 大资金无法执行 |
| 6 | **因子独立性** | 发现的因子是否独立于已知因子？ | 否 = 只是已知因子的变体 |
| 7 | **稳健性** | 参数微调后策略是否仍然有效？ | 否 = 策略过于脆弱 |

## 深度分析框架

### 第1步：量化思维

**西蒙斯的核心原则：**
- 不依赖直觉或故事，依赖数据和统计
- 寻找市场中的统计异常（定价错误）
- 用小而确定的优势积累收益
- 分散化是免费午餐

**量化分析步骤：**
1. **假设形成**：基于观察或理论提出可检验的假设
2. **数据收集**：收集足够大的样本（至少100+数据点）
3. **统计检验**：用适当的统计方法检验假设
4. **样本外验证**：用未参与建模的数据验证
5. **交易成本调整**：扣除佣金、滑点、冲击成本
6. **风险评估**：VaR、最大回撤、夏普比率

### 第2步：因子分析

**常见有效因子：**
| 因子 | 逻辑 | 预期方向 |
|------|------|---------|
| 价值(PE/PB/EV) | 便宜股票长期跑赢 | 低估值→高回报 |
| 动量 | 涨的继续涨，跌的继续跌 | 过去12个月涨幅→未来3-6个月回报 |
| 质量(ROE/ROIC) | 好公司长期跑赢 | 高ROIC→高回报 |
| 低波动 | 低波动股票风险调整后回报更高 | 低波动→高夏普比率 |
| 规模 | 小盘股长期跑赢大盘股 | 小市值→高回报 |
| 盈利增长 | 盈利增长加速的股票跑赢 | 增速加快→高回报 |

**因子独立性检验：**
- 新发现的因子是否与已知因子相关？
- 如果相关，控制了已知因子后是否仍有超额收益？
- 因子是否有经济逻辑支撑？

### 第3步：统计套利

**配对交易：**
- 找到历史上价格高度相关的两只股票
- 当价差偏离历史均值时入场
- 价差回归均值时出场
- 关键：配对必须基于经济逻辑（同行业、同供应链）

**多因子模型：**
```
预期收益 = α + β1×价值 + β2×动量 + β3×质量 + ... + ε
```
- 用回归分析估计各因子的权重
- 选择因子暴露最优的股票组合
- 定期再平衡（月度/季度）

### 第4步：风险管理

**西蒙斯式的风险控制：**
| 风险指标 | 计算方法 | 警戒线 |
|---------|---------|--------|
| 夏普比率 | (策略收益-无风险利率)/策略波动率 | <1.0 需改进 |
| 最大回撤 | 策略最大峰谷跌幅 | >20% 需审查 |
| VaR(95%) | 95%置信度下的最大日损失 | 根据风险偏好 |
| 因子暴露 | 对各因子的敏感度 | 避免过度集中于单一因子 |
| 相关性 | 组合内资产的平均相关性 | 越低越分散 |

### 第5步：模型迭代

**模型生命周期：**
1. **研发阶段**：发现新的统计规律
2. **回测阶段**：用历史数据验证（注意过拟合陷阱）
3. **模拟阶段**：用实时数据模拟运行
4. **实盘阶段**：小资金实盘验证
5. **扩大阶段**：验证有效后扩大资金
6. **监控阶段**：持续监控模型表现
7. **衰减处理**：模型失效时及时替换

**过拟合防范：**
- 使用交叉验证
- 限制模型参数数量
- 要求样本外表现与样本内一致
- 用经济逻辑解释统计发现
