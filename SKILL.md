## 六层分析框架总览

| 层次 | 核心问题 | 对应大师视角 | 权重 |
|------|---------|------------|------|
| **志** | 我的投资信仰是什么？心性是否成熟？ | 格雷厄姆(安全边际信仰) + 巴菲特(价值投资) + 段永平(本分) | 心性基石 |
| **道** | 这笔投资的底层逻辑是什么？ | 巴菲特(能力圈) + 芒格(心智模型) + 格雷厄姆(内在价值) | 定性基石 |
| **势** | 市场趋势和周期处于什么阶段？ | 索罗斯(反身性) + 马克斯(周期) + 达利欧(债务周期) | 时机判断 |
| **法** | 如何系统化地判断值不值得投？ | 林奇(PEG) + 费雪(15点) + 格雷厄姆(定量筛选) | 流程+方法 |
| **术** | 具体怎么操作？仓位如何管理？ | 索罗斯(仓位策略) + 十大师买卖标准 | 技术+执行 |
| **器** | 用什么工具和技术来辅助决策？ | 西蒙斯(量化模型) + 技术指标 + 算法 | 工具+量化 |

六层关系：**志→道→势→法→术→器**
- **志**决定信仰和心性 — 没有正确的投资信仰，一切方法论都是空中楼阁
- **道**决定方向 — 方向错了，其余再精妙也是南辕北辙
- **势**决定时机 — 好公司也需要好时机
- **法**提供方法论 — 没有流程的投资是赌博
- **术**保障执行 — 好的分析需要好的执行才能变现
- **器**提升效率 — 工具让分析更精确，但不能替代判断

上层错了，下层再精妙也是徒劳。

## 报告输出规范

每次完成分析后，必须将完整报告以 Markdown 格式保存到本地文件系统。

### 文件名格式

```
~/.stock-invest-master/yyyymmdd_X公司.md
```

- `yyyymmdd`：分析日期，如 20260509
- `X公司`：公司简称（中文或英文），如 腾讯、苹果、茅台、Apple
- 示例：`~/.stock-invest-master/20260509_腾讯.md`
- 同日同公司多次分析，自动追加版本号：`20260509_腾讯_v2.md`

### 保存要求

1. **目录自动创建**：如果 `~/.stock-invest-master/` 不存在，自动创建
2. **报告完整性**：保存完整分析报告，包含六层评估、十大师共识、违背诊断、监控指标等所有章节
3. **编码格式**：UTF-8，确保中文字符正确显示
4. **快速判断路径**：路径A（快速筛选）的报告也需保存，文件名追加 `_quick` 后缀

### 保存时机

- 完整分析（路径B）：六层评估和十大师共识全部完成后保存
- 快速筛选（路径A）：快速筛选完成后保存
- 补充分析：任何后续追加分析都保存到新版本文件

### 保存后自动启动 Web 服务

**每次保存报告后，必须执行以下命令确保 Web 浏览服务正常运行：**

```bash
bash ~/.hermes/skills/stock-invest-master/scripts/manage_server.sh ensure
```

该脚本会自动检查服务是否在 8888 端口运行，如未运行则启动。服务后台常驻，支持：
- `ensure` — 检查并自动启动（报告保存后必调）
- `start` — 手动启动
- `stop` — 停止服务
- `restart` — 重启服务
- `status` — 查看运行状态

健康检查接口：`http://127.0.0.1:8888/health`

## 十位投资大师

| 大师 | 核心方法论 | 一句话精髓 | 对应层次 |
|------|-----------|-----------|---------|
| 📚 **格雷厄姆** (Benjamin Graham) | 安全边际 + 内在价值 + 定量分析 | "市场短期是投票机，长期是称重机" | 志·道·法 |
| 🎩 **巴菲特** (Warren Buffett) | 能力圈 + 护城河 + 安全边际 | "风险来自不知道自己在做什么" | 志·道 |
| 🏃 **林奇** (Peter Lynch) | 六类分类 + PEG + 两分钟演练 | "买你了解的" | 法 |
| 🔬 **费雪** (Philip Fisher) | 15点选股 + 闲聊法 + 长期持有 | "如果买对了，几乎永远不卖" | 法 |
| 🧠 **芒格** (Charlie Munger) | 逆向思维 + 心智模型格栅 + 认知偏差 | "以合理价格买伟大生意" | 道 |
| 📊 **马克斯** (Howard Marks) | 周期定位 + 第二层次思维 + 风险聚焦 | "最危险的事是相信没有风险" | 势 |
| 🎯 **段永平** | 本分哲学 + Stop Doing List + 睡得着觉 | "做对的事情，把事情做对" | 志·道 |
| 🌊 **达利欧** (Ray Dalio) | 经济机器 + 债务周期 + 全天候策略 | "理解因果，系统化决策" | 势 |
| 💥 **索罗斯** (George Soros) | 反身性理论 + 宏观对冲 + 泡沫识别 | "世界经济史是一部基于假象和谎言的连续剧" | 势·术 |
| 🤖 **西蒙斯** (James Simons) | 量化模型 + 统计套利 + 机器学习 | "市场不是不可预测的，只是需要正确的模型" | 器 |

不是套公式——而是像10位大师同时在场，各抒己见，最终形成共识。

> "市场短期是投票机，长期是称重机。" —— 格雷厄姆
>
> **注入大师视角**：格雷厄姆(安全边际信仰) + 巴菲特(价值投资) + 段永平(本分)

"志"是投资体系的地基，决定了你是什么样类型的投资者。没有正确的投资信仰，一切方法论都是空中楼阁。

### 1.1 投资信仰检验

## 第四层：法 — 方法论与系统化流程

> "用合理价格买伟大生意，胜过用便宜价格买平庸生意。" —— 芒格
>
> **注入大师视角**：林奇(PEG) + 费雪(15点) + 格雷厄姆(定量筛选)

### 4.1 商业模式与财务分析

| 维度 | 核心指标 | 健康标准 | 红旗 |
|------|---------|---------|------|
| 盈利能力 | ROE(>15%), ROIC(>WACC), 毛利率(稳定) | 持续优秀 | ROE<10%或大幅波动 |
| 成长性 | 营收增长率, 利润增长率, FCF增长 | 持续正增长 | 增收不增利 |
| 安全性 | 资产负债率(<60%), 流动比率(>1.5), CFO/净利润(>0.9) | 稳健 | 负债率>80% |
| 效率 | 应收/存货周转天数, 总资产周转率 | 稳定或改善 | 持续恶化 |

### 4.2 估值方法体系

| 公司类型 | 首选方法 | 辅助方法 | 不适用方法 |
|---------|---------|---------|-----------|
| 稳定增长型 | DCF | 可比公司(PE/EV/EBITDA) | PS |
| 高速成长型 | PEG, PS | DCF(多场景) | PE(如未盈利) |
| 周期型 | normalized PE, EV/EBITDA | PB | DCF |
| 资产重型 | PB, NAV | EV/EBITDA | PE, PS |
| 价值型(格雷厄姆) | 净流动资产价值 | PB, PE | DCF |
| SaaS/订阅制 | PS, Rule of 40 | DCF(LTV/CAC) | PE, PB |

**多方法交叉验证**：至少2种独立方法，差异>50%时必须解释原因。

### 4.3 大师视角注入

| 巴菲特 | Owner Earnings？ROIC>15%？护城河趋势？ |

### 4.4 标准化研究流程

```

Step 2: 商业模式理解（收入拆解/成本结构/价值链/护城河）
```

### 5.1 择时策略

| 策略 | 适用场景 | 具体方法 |
|------|---------|---------|
| 分批建仓 | 看好但价格不够便宜 | 3-5批，每批间隔2-4周或下跌5-10% |
| 定投策略 | 长期看好，短期不确定 | 固定周期/金额 |

| 索罗斯式 | 反身性转折确认 | 小仓位试错，确认后加码 |

### 5.2 仓位管理

**金字塔建仓法：**
```
第一批（试探）: 总计划仓位的 20-30%
    ↓ 确认判断正确 / 价格更具吸引力
第二批（加仓）: 总计划仓位的 30-40%
    ↓ 基本面验证 / 趋势确认
第三批（重仓）: 总计划仓位的 30-40%
```

**仓位控制原则：**
- 单只股票 ≤ 总仓位的 10-15%（高确定性可达20%）
- 单一行业 ≤ 总仓位的 30%
- 现金仓位 ≥ 10%（保持灵活性）
- 索罗斯式：试错仓位小，确认后敢于重仓

## 多大师深度分析框架（注入到六层中）

### 第一层：志（格雷厄姆+巴菲特+段永平主导）

> "投资的核心是在安全边际内购买内在价值。" —— 格雷厄姆

1. **安全边际信仰检验**：是否真正理解"价格≠价值"？（格雷厄姆）
2. **价值投资信仰**：是否愿意在恐慌中买入？（巴菲特）
3. **本分心态检验**：投资是否让你"睡得着觉"？（段永平）
4. **投机vs投资辨别**：当前行为是投资还是投机？（格雷厄姆）

### 第二层：道（巴菲特+芒格+格雷厄姆主导）

> "如果说不清它靠什么赚钱，就不要买。" —— 巴菲特

1. **能力圈检验**：能否一段话说清赚钱方式？（巴菲特）
2. **护城河判断**：五种护城河类型+趋势（巴菲特）
3. **内在价值计算**：是否可合理估算？（格雷厄姆）
4. **心智模型覆盖**：能用多模型分析吗？（芒格）
5. **定价权测试**：能否涨价不流失客户？（巴菲特+芒格+段永平）

### 第三层：势（索罗斯+马克斯+达利欧主导）

> "市场总是错的，但错的方向可能持续很久。" —— 索罗斯

1. **反身性分析**：主流叙事是什么？反馈循环方向？（索罗斯）
2. **钟摆定位**：市场处于贪婪还是恐惧？（马克斯）
3. **债务周期**：短/长周期什么阶段？（达利欧）
4. **第二层次思维**：共识是什么？为什么可能错？（马克斯）
5. **全天候检验**：增长/通胀四象限存活能力？（达利欧）

### 第四层：法（林奇+费雪+格雷厄姆主导）

> "买之前，先搞清楚这是什么类型的股票。" —— 林奇

1. **格雷厄姆定量筛选**：PE<15？PB<1.5？净流动资产>市值？（格雷厄姆）
2. **林奇六类分类**：归属哪一类？（林奇）
3. **费雪15点评分**：商业/盈利/人的因素/竞争地位（费雪）
4. **PEG决策**：PEG<0.5极度吸引 → PEG>2.0过贵（林奇）
5. **安全边际计算**：内在价值 vs 当前价的差距（格雷厄姆+巴菲特）

### 第五层：术（索罗斯+十大师标准）

> "重要的不是你对还是错，而是对的时候赚多少，错的时候亏多少。" —— 索罗斯

1. **索罗斯仓位策略**：小仓位试错→确认后加码（索罗斯）
2. **金字塔建仓**：分批建仓的节奏控制
3. **十大师卖出标准**：逐项检查卖出条件
4. **止损/止盈**：明确的退出计划

### 第六层：器（西蒙斯+工具辅助）

> "我们不需要理解市场，只需要找到统计规律。" —— 西蒙斯

1. **量化验证**：数据一致性校验、历史分位分析
2. **技术指标**：辅助判断（非主导）
3. **回测验证**：用历史数据验证结论
4. **压力测试**：蒙特卡洛模拟最坏情况

## Common Pitfalls

1. **志不牢，一切方法论都是空中楼阁。** 没有正确的投资信仰，技术分析再精妙也只是高级赌博。
2. **重术轻志。** 最危险的错误是从"器"开始分析（看K线、追热点），而不先问"志"和"道"。
3. **把"法"等同于看几个财务指标。** "法"是对商业模式本质的理解+系统化流程，不是简单的数字对比。
4. **忽视"势"的力量。** 好公司+好价格，如果处于泡沫末期或衰退初期，也可能面临长期套牢。
5. **"器"层越位。** 量化模型和技术指标在投资中只起辅助作用，不能替代基本面判断。
6. **忽视索罗斯的反身性。** 当市场叙事形成自我强化循环时，基本面分析可能暂时失效。
7. **忽视西蒙斯的统计视角。** 任何投资结论都应该有数据支撑，而非直觉。
8. **静态分析。** 所有分析都要考虑趋势——ROE改善还是恶化？护城河变宽还是变窄？
9. **自我欺骗。** 一旦看好，容易选择性收集支持信息。务必列出3条可能亏钱的路径。
10. **对红旗信号视而不见。** 当发现财务造假、管理层失信等自动否决项时，仍不愿放弃——这是沉没成本偏差。
